# HizenLabs.Plugins

Free, open-source plugins for [Rust](https://rust.facepunch.com/) servers — built on
[Carbon](https://carbonmod.gg/) and compatible with [Oxide/uMod](https://umod.org/).

Every plugin ships as a single drop-in `.cs` file. No extra dependencies to install,
no extensions to manage — grab the one you want, drop it in, done.

## Plugins

## Installing a plugin

1. Download the plugin's `.cs` file from this repo (or from [Releases](https://github.com/HizenLabs/HizenLabs.Plugins/releases)).
2. Drop it into your server's plugins folder:
   - **Carbon:** `carbon/plugins/`
   - **Oxide:** `oxide/plugins/`
3. The server compiles and loads it automatically. Settings appear under
   `carbon/configs/` or `oxide/config/` for you to tweak.
